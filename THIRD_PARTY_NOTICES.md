# Third-party Notices

Status: canonical

## OpenTaiko

OpenTaiko is a third-party reference project. This repository does not contain copied OpenTaiko source files, bundled song packages, bundled skin packages, or game assets. OpenTaiko-compatible behavior is documented as interoperability research and acceptance criteria.

## Rust dependencies

Rust dependency notices are derived from `Cargo.lock` and package metadata during release packaging. During the operations migration rail, dependency notice generation is a deterministic CI artifact rather than a committed vendored dependency tree.

## User-selected assets

User-selected charts, audio, images, videos, and skins are not repository content. Local validation manifests may point to files outside the repository, but the files themselves must remain untracked.
