# QA Reports

This directory is reserved for machine-readable QA / Regression Session evidence.

Expected generated files include:

- `phase1_loop.qa.json` from `taiko_cli qa run --manifest fixtures/synthetic/phase1_synthetic_manifest.toml --threshold-ms 1.0 --format json`
- `phase1_loop.compare.json` from `taiko_cli qa compare --baseline reports/baseline --current reports/current --format json`
- `phase1_loop.verdict.json` from `taiko_cli qa verdict --input reports/qa/phase1_loop.qa.json --format json`

Generated QA evidence is intentionally not pre-populated in the bootstrap package. Gate reports must be generated by the QA / Regression Session in a separate worktree.
