# E2E Smoke Loop Reports

Status: canonical

This directory is reserved for Step22 E2E smoke loop evidence generated by:

```bash
scripts/run_e2e_smoke_loop.sh --scenario all --dry-run
```

Committed packages keep only this README. Smoke scenario output is generated by CI, Codex Cloud, Codex App Automations, or local operator runs.

Smoke output must not include user-selected song assets and must not create live GitHub PRs unless the operator explicitly removes `--dry-run` for a dedicated controller test.
