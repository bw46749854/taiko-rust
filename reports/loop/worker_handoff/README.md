# Worker Handoff Reports

Status: canonical

This directory stores deterministic Codex worker handoff artifacts emitted by GitHub Actions or local static checks.

Canonical latest files:

```text
latest.json
latest.md
latest_issue.md
latest_comment.md
```

These files are generated by:

```bash
scripts/loop_emit_worker_handoff.py --mode plan --run-id RUN-HANDOFF-LOCAL
```

The generator does not call Codex, GPT, OpenAI APIs, or `openai/codex-action@v1`. GitHub Actions uploads this directory as controller evidence.
